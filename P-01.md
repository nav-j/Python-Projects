## Scientific Calculator using Python Tkinter

##  Features

### Basic Calculator Functions
- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🟰 Evaluate Expressions
- ⌫ Backspace
- 🧹 Clear Screen
- Parentheses Support `( )`
- Decimal Numbers

### Scientific Functions
- √ Square Root
- x² Square
- xʸ Power Function
- Factorial (!)
- sin()
- cos()
- tan()
- log₁₀()
- Natural Log (ln)
- π (Pi)
- e (Euler's Number)

- Python 3.x
- Tkinter (GUI)
- Math Module

```bash

##  Interface

The calculator includes:

- Display Screen
- Scientific Function Buttons
- Number Buttons
- Arithmetic Operators
- Backspace Button
- Clear Button
- Pi and Euler Constant Buttons

---

##  Function Description

| Function | Description |
|----------|-------------|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `=` | Evaluate Expression |
| `⌫` | Delete Last Character |
| `C` | Clear Display |
| `√` | Square Root |
| `x²` | Square of Number |
| `xʸ` | Power (Base,Exponent) |
| `!` | Factorial |
| `sin` | Sine (Degrees) |
| `cos` | Cosine (Degrees) |
| `tan` | Tangent (Degrees) |
| `log` | Base-10 Logarithm |
| `ln` | Natural Logarithm |
| `π` | Insert Pi Value |
| `e` | Insert Euler's Number |

```

### Trigonometric Function

```
Input:
30

Press:
sin

Output:

```

---

##  Notes

- Trigonometric functions accept input in **degrees**.
- The **Power (xʸ)** function requires input in the format:

```
Base,Exponent
```

Example:

```
2,5
```

- Invalid inputs display an error message.
